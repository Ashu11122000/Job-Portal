## About The Project
This job portal lets users (candidates) register and browse/post jobs, apply for jobs, and lets companies manage job postings.  
The system is built with a React frontend, a Node.js/Express backend and a MySQL (or other relational) database.  
It includes upload capability (e.g., resumes), role-based access (candidate / employer / admin), job management, application tracking, notifications etc.

## Built With
- Frontend: React, JavaScript, Tailwind CSS or other UI  
- Backend: Node.js, Express.js  
- Database: MySQL (or your choice)  
- Containerization: Docker, Docker-Compose, Kubernetes (Minikube)  
- Additional: REST APIs, JWT Authentication, Role & Permission management, File Upload (Cloudinary or local), etc

## Features
- Candidate registration & login  
- Company/employer registration & login  
- Post & manage job postings (by employer)  
- Browse jobs & apply (by candidate)  
- View and manage applications (by employer)  
- Admin dashboard for analytics and oversight  
- File upload for resumes/profiles  
- Notification system  
- Role-based access control (admin / employer / candidate)  
- Dockerized setup for easier local dev & deployment
